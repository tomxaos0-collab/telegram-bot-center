# Changelog

## 0.1.0

*   Добавлено desktop-приложение Telegram Bot Center на базе Tauri.
*   Управление несколькими Telegram-ботами.
*   Поддержка проверки `getMe` и получения обновлений `getUpdates`.
*   Автоматическое определение `chat_id` и `message_thread_id` (topics).
*   Интерфейс управления Webhook (установка, удаление, статус).
*   Система маршрутов уведомлений для различных типов событий.
*   Модуль HTML-шаблонов с Live Preview и автоматическим определением переменных.
*   Возможность тестовой отправки сообщений по шаблону через маршруты.
*   Локальное хранилище данных SQLite.
*   Автоматическая проверка обновлений через GitHub.
