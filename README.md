# Telegram Bot Center

Локальное desktop-приложение под Linux для управления Telegram-ботами, чатами, webhook, маршрутами уведомлений и HTML-шаблонами сообщений.

## Возможности

*   **Управление несколькими Telegram-ботами**: Добавление и переключение между неограниченным количеством ботов.
*   **Проверка бота через getMe**: Мгновенная проверка статуса и токена.
*   **Получение updates**: Чтение последних входящих обновлений от Telegram API.
*   **Определение chat_id и topic_id**: Автоматический парсинг метаданных из входящих сообщений.
*   **Управление webhook**: Установка, удаление и просмотр информации о вебхуке.
*   **Маршруты уведомлений**: Настройка направлений для системных уведомлений (Z-отчёты, ошибки, инкассации и т.д.).
*   **Шаблоны Telegram HTML-сообщений**: Визуальный редактор с поддержкой переменных и Live Preview.
*   **Тестовая отправка сообщений**: Проверка шаблонов через любой настроенный маршрут.
*   **Экспорт ENV/JSON**: Удобное копирование конфигураций для вашего проекта без раскрытия секретов.

## Установка на Linux / KachiOS / Arch

Для работы приложения и его сборки необходимо установить системные зависимости:

```bash
sudo pacman -S --needed base-devel curl wget file openssl webkit2gtk-4.1 librsvg rust npm
```

## Запуск в dev-режиме

1.  Установите зависимости Node.js:
    ```bash
    npm install
    ```
2.  Запустите Tauri в режиме разработки:
    ```bash
    npm run app:dev
    ```

## Сборка

Для создания исполняемого файла вручную выполните:

```bash
npm run app:build
```

Бинарный файл после сборки будет находиться по пути:
`src-tauri/target/release/telegram-bot-center`

## Локальная установка

Для автоматической сборки, копирования бинарника и создания ярлыка в меню приложений выполните команды:

```bash
chmod +x install.sh uninstall.sh
./install.sh
```

Или через npm:
```bash
npm run install:local
```

После установки приложение появится в меню KDE/GNOME (или на рабочем столе) как **Telegram Bot Center**.

## Обновление

После того, как вы скачаете (или напишете) новые изменения в коде, просто запустите:

```bash
./install.sh
```

Скрипт сам закроет старую версию, соберёт новую, заменит бинарный файл и обновит ярлык.

## Удаление

Для удаления установленного приложения:

```bash
npm run uninstall:local
```

**Важно:** Скрипт удаления стирает только бинарные файлы и ярлыки. Ваши пользовательские данные (база данных SQLite, добавленные боты, токены) не удаляются автоматически.

## Безопасность

*   **НЕ ПУБЛИКУЙТЕ** ваш Telegram bot token в открытых источниках.
*   Не храните реальные токены в Git (они должны вводиться только в интерфейсе приложения).
*   Локальный экспорт ENV/JSON не содержит в себе токенов (только ссылки на них или юзернеймы).
*   Если токен случайно попал в публичный репозиторий — **немедленно** перевыпустите его через @BotFather.
*   Приложение является полностью локальным. Ваши данные (токены, чаты) хранятся в зашифрованном/защищенном виде (в будущем) или локальной SQLite базе на вашем компьютере.
